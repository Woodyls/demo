# 五子棋小程序

一个纯前端（HTML/CSS/JS）的五子棋小游戏，支持：
- 15x15 棋盘渲染
- 黑白棋轮流落子
- 五连胜自动判定
- 高亮最后一手
- 悔棋与重新开始

## 使用方式

- 直接用浏览器打开 `index.html` 即可离线游玩。
- 或在本地启动静态服务器：
  - Python：`python -m http.server 8000`，然后访问 `http://localhost:8000/`。
  - Node（需安装）：`npx serve .` 或 `npx http-server -p 8000`。

## 操作说明

- 鼠标点击棋盘落子；黑棋先手。
- 点击“悔棋”撤销上一手。
- 点击“重新开始”清空棋盘重新开局。
