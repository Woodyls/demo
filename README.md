# 五子棋小程序

一个纯前端（HTML/CSS/JS）的五子棋小游戏。

## 功能特性

- 棋盘与交互：15×15（支持 13/15/19 路数切换）、黑白轮流、五连胜自动判定、平局判定、悔棋与重新开始。
- 可视优化：悬停虚影、最后一手高亮、棋盘边缘坐标标注（A–O, 1–15）。
- 交互增强：落子音效与最后一手箭头提示（均可开关）。
- 棋谱能力：
  - 导出 JSON/TXT，支持复制到剪贴板；文件命名包含路数与时间戳（例如 `gomoku-15-20251108-142305.txt`）。
  - 导入棋谱（JSON/TXT），带格式/越界/重复坐标校验并错误提示。
- 着法标记：勾选“显示手数”在棋子中心显示着法序号。
- 主题与路数：浅色/深色主题切换；13/15/19 路数切换并自适应星位。

## 使用方式

- 直接用浏览器打开 `index.html` 即可离线游玩。
- 或在本地启动静态服务器：
  - Python：`python -m http.server 8000`，然后访问 `http://localhost:8000/`。
  - Node（需安装）：`npx serve .` 或 `npx http-server -p 8000`。

## 操作说明

- 落子：鼠标点击或触控点击棋盘；黑棋先手。
- 悔棋/重开：点击“悔棋”“重新开始”。
- 着法标记：在底部控制区勾选“显示手数”。
- 导出棋谱：使用“导出JSON”“下载TXT”“复制TXT”。
- 导入棋谱：
  - JSON 示例：`{"grid":15,"moves":[{"r":7,"c":7,"player":1}]}`，导入时会校验 `grid` 为 13/15/19 并按 `grid` 切换路数。
  - TXT 示例：每行一手，如 `黑 A1` 或 `W H8`。导入时会校验坐标格式、越界与重复坐标；错误将以红色状态提示首个错误行与错误总数。
- 主题与路数：在控制区切换“深色主题”复选框与“路数”下拉框（13/15/19）。
- 落子音效：在控制区勾选“落子音效”开关以打开/关闭提示音。
- 箭头提示：在控制区勾选“箭头提示”开关以显示/隐藏最后一手箭头。
- 偏好持久化：音效/箭头开关、显示手数、深色主题与路数选择会自动保存在浏览器 `localStorage` 中，刷新后仍保留。
